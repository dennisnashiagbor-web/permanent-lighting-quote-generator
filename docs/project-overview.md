# Project Overview

This project was built as a real-world quoting and proposal system for a permanent lighting business.

The main objective was to create a more professional and efficient quoting workflow that could replace manual pricing methods and improve the customer-facing sales process.

## Project Goal

The goal was to build a lightweight tool that could help a business:

- estimate roofline footage
- calculate project pricing consistently
- present a polished customer proposal
- support promotional pricing
- improve the sales workflow

## Business Context

Many small service businesses still rely on spreadsheets, handwritten estimates, or generic templates to create customer quotes. That often leads to slower turnaround times, inconsistent pricing, and a less polished customer experience.

This project was designed to solve that problem through a simple browser-based workflow.

## Key Product Decisions

Important design and implementation decisions included:

- using a browser-based format for accessibility
- creating a printable proposal layout
- using interactive mapping for roofline estimation
- incorporating discount logic directly into the pricing workflow
- improving layout consistency for customer presentation
- making the final output suitable for customer review and PDF export

## Functional Capabilities

The final tool supports:

- property search
- map-based roofline tracing
- footage calculation
- pricing inputs
- discount handling
- proposal generation
- property image placement
- digital signatures
- print/PDF-ready formatting

## Outcome

The final result was a lightweight, practical quoting tool that demonstrates front-end development, workflow design, automation thinking, and business problem-solving in a real-world setting.

## Why This Project Matters

This project is a strong example of how technical skills can be applied beyond dashboards and analysis. It shows the ability to identify a workflow problem, define requirements, iterate on a solution, and produce a usable customer-facing product.
